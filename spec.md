# Specification

## Summary
**Goal:** Create a modern landing page for Trouthstem with a warm, earth-toned design aesthetic.

**Planned changes:**
- Build a single-page landing site with header navigation, hero section, and about section
- Implement a cohesive visual design using warm earth tones (terracotta, sage green, cream)
- Create responsive navigation menu with links to main sections
- Add clear typography hierarchy for headings and body text

**User-visible outcome:** Users can visit the Trouthstem landing page, navigate between sections, and learn about the brand through the about section in a modern, visually appealing layout.
